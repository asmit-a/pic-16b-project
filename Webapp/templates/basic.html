Family Demographics

The data displayed on this web app is based on the US Census's American Community Survey 5-year estimates, which is an ongoing survey that provides annual estimates based on data collected over 5-year periods. Using multiyear estimates allows for increased statistical reliability of data, particularly for regions and subgroups that have smaller populations.
